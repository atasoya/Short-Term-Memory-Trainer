Build 
===========
Start a new [flutter](https://flutter.dev/docs/get-started/test-drive?tab=terminal) app

```
flutter create shorttermmemorytraning
```
<br>

Change your lib folder , pubspec.yaml file and add assets folder.

<br>

Now run the app

```
flutter run 
```
